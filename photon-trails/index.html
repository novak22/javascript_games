How to navigate the trails

  1. Select any glowing tile to rotate it 90° and change how it bends or splits the beam.
  2. Chain emitters, mirrors, and splitters so the photon flow touches every crystal.
  3. Watch the board update in real time—when all crystals are lit the run is complete.

Photon Trails supports mouse, touch, and keyboard play. Use Tab to focus a tile and press Enter or Space to rotate it. Status updates announce crystal progress for assistive tech.
